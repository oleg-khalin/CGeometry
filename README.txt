	Виконавець: ліцеїст III-A курсу Халін Олег Ігорович.
	Керівники роботи: Сотніченко Олена Віталіївна та Боровик Людмила Іванівна.
	Тема: Обчислювальна геометрія на площині.
	Дана тема була видбрана через бажання зробити корисну програму, яка б стала в нагоді вчителям, які хочуть наочно продемонструвати способи вирішення задач
обчислювальної геометрії на площині, та учням, які вивчають тему самостійно та прагнуть її краще зрозуміти, також для олімпіадників при підготовці до олімпіад
високого рівня.
	Мета: Основною метою роботи було дослідити методи вирішення задач обчислювальної геометрії та змоделювати деякі з них. Для досягнення цієї мети було
поставлено наступні задачі:
1.	Побудова опуклої оболонки для множини з N точок площини;
2.	Перевірка опуклості многокутника;
3.	Перевірка приналежності точки внутрішньої області многокутника;
4.	Коло, що «охоплює» N точок площини.
	Мова програмування: C#.
	Середовище: Visual Studio 2013-2015.

	Апаратні вимоги:
Для використовування програми рекомендується:
•	Процесор AMD Dual-Core C-60 або краще;
•	2 GB оперативної пам’яті;
•	Маніпулятор – «миша»;
•	Монітор з роздільною здатністю 1366 х 768;
•	20 MB вільного місця на жорсткому диску;
•	Підключення до Інтернету.

	Програмні вимоги
Для використовування програми є необхідними:
•	Операційна система Windows XP/ 7/ 8/10;
•	Браузер Google Chrome/ Opera/ Mozilla Firefox /Internet Explorer з підтримкою WebGL;
•	.NET Framework 4.5.

Файл запуску: CGeometry.exe.
